# Customer Behavior Analysis & Power BI Dashboard

This project focuses on analyzing customer behavior and sales performance using SQL, Python, and Power BI. The dataset was queried and processed using SQL, cleaned and explored in Python, and finally visualized through an interactive Power BI dashboard to deliver actionable business insights.

The dashboard highlights key metrics such as customer distribution, subscription status, revenue by category, sales by age group, and purchasing patterns, enabling data-driven decision-making for marketing and business strategy.

# Key highlights:
Data extraction and analysis using SQL

Data cleaning and exploratory analysis using Python

Interactive Power BI dashboard for business insights

Customer segmentation and sales trend analysis

# Tools & Technologies:
SQL, Python (Pandas, NumPy), Power BI, Jupyter Notebook
